# Supply-chain-analysis
Power BI Data Cleaning &amp; Extraction Pipeline  This project demonstrates an end-to-end data preparation workflow using SQL and Python to clean, transform, and extract raw business data for Power BI reporting and dashboards.  The solution focuses on applying industry best practices such as data validation.
The cleaned datasets are designed to support key business KPIs including sales performance, profitability, operational efficiency, and trend analysis, enabling faster insights and accurate decision-making in Power BI.

Cleaned and standardized raw transactional data using SQL (CTEs, window functions, views)
